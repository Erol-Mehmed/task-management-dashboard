# Project Name

Task Management Dashboard

A web application for managing tasks.

## Tech stack

- TypeScript
- Vue
- Node.js
- npm

## Prerequisites

- Node.js v16+ (use `nvm` to manage versions)
- npm v8+ (or yarn/pnpm)

## Install

````bash
npm install

## Run

```bash
npm run dev
````

## API

The project uses https://mockapi.io/ for simulating backend API calls.
